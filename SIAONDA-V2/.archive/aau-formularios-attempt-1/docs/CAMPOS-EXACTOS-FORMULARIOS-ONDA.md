# CAMPOS EXACTOS DE FORMULARIOS ONDA (Según PDFs Oficiales)

## NOTA IMPORTANTE
Todos los formularios comparten los mismos **DATOS DEL AUTOR** (Sección 1):
- Nombre
- Apellido
- Seudónimo
- Nacionalidad
- Documento (Cédula/Pasaporte/RNC)
- Provincia
- Municipio
- Calle
- Sector
- Número de vivienda
- Correo electrónico
- Teléfono residencial
- Teléfono móvil
- ¿Es obra póstuma? (checkbox)
- Fecha de fallecimiento (si aplica)

**ESTOS DATOS SE MANEJAN EN LA TABLA DE CLIENTES, NO EN CAMPOS DINÁMICOS**

## OBSERVACIONES IMPORTANTES:

1. **TODOS los formularios tienen un campo de "Carácter de la obra"** con opciones similares
2. **TODOS tienen campos para obras derivadas** con nombre del autor original
3. **TODOS piden descripción breve** al final
4. **Los datos del autor/cliente NO van en campos dinámicos**, van en la tabla `clientes`
5. **Los datos del solicitante (si no es el autor) NO van en campos dinámicos**, son opcionales

## CAMPOS COMUNES QUE NO DEBEN SER DINÁMICOS:
- Fecha de solicitud → fecha del formulario
- Firma del solicitante → campo `firma` en tabla formularios
- Número de formulario → auto-generado como código
- Anexar copia de cédula → campo archivo, pero no es campo dinámico de obra

## TIPOS DE CAMPO A USAR:
- `texto`: Para campos cortos (título, género, país, etc.)
- `textarea`: Para descripciones, listados largos
- `numerico`: Para años, cantidades
- `fecha`: Para fechas específicas
- `checkbox`: Para opciones múltiples (carácter, tipo)
- `radio`: Para opciones únicas (categoría de audiovisual)
- `archivo`: NO se definen como campos de formulario, se suben aparte
